Meteor Atmosphere packages are a built-in ecosystem used only in Meteor. They have been useful since the beginning because they integrate directly with Meteor’s lifecycle, acting on changes at build time and runtime. Many community packages already exist, adding third-party functionality and helping developers focus on product features.
